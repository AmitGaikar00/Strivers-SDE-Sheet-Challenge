# Majority Element II #

Given an integer array of size n, find all elements that appear more than ⌊ n/3 ⌋ times.

 

Example 1:
<br/>
Input: nums = [3,2,3]
<br/>
Output: [3]
<br/>

Example 2:
<br/>
Input: nums = [1]
<br/>
Output: [1]
<br/>

Example 3:
<br/>
Input: nums = [1,2]
<br/>
Output: [1,2]
<br/> 

Constraints:
<br/>
1 <= nums.length <= 5 * 104
<br/>
-109 <= nums[i] <= 109
